# Thockpit

A typing test you can hear. A 3D mechanical keyboard sits under the words and
reacts as you type — the right keycap sinks, the next one glows, and every stroke
plays a recording of a real switch.

```bash
npm install
npm run dev     # http://localhost:3000
```

## What's in it

**The board** — a MacBook (US ANSI) layout rendered in WebGL with three.js.

**The sound** — recordings of real switches, not synthesis. Four packs ship.

**The light** — pull the cord in the top right to switch the theme.

**The stats** — a graph of speed per second, saved to localStorage.
